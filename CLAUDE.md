# Project Rules — GuestChat Landing Page

## Verification
- **The user checks every change manually.** Do not spend effort auto-verifying UI changes with the preview server, screenshots, or `preview_*` tools. After making a change, briefly state what changed and let the user verify it in their own browser.
- Only run/verify in the browser if the user explicitly asks for it.

## Content & Copy
- **No emoji anywhere in this project.** Not in UI copy, mock/demo content, component data, code comments, or commit messages. Use plain text, typographic glyphs (e.g. `★ ✓ ✕ →`), or SVG icons instead of emoji.
